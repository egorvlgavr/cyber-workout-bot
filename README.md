# cyber-workout-bot
Telegram bot for workout plans
 - Base exercises
 - Isolated exercises
 - Additional exercises

# How to run
Python 3.11 is required. 

All dependencies are placed in requirements.txt

Env variables
```
DATA_PATH=/data/workouts.json
TELEGRAM_TOKEN={YOUR_TOKEN}
```
Run from command line
```commandline
python3 bot.py
```
